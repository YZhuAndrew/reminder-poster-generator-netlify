# 更新说明（v2 · 视觉重新设计）

本次更新对海报生成器进行了全面的视觉与编辑体验重新设计。以下变更已推送至线上，刷新页面即可看到新版。

---

## 🎨 海报视觉：全面升级为「暖纸编辑风」

告别旧的渐变底 + 卡片堆叠式版面，改为更克制、更高级的编辑设计语言：

- **纯色暖纸底**：去掉所有廉价渐变与假纹理，改为温润的纯色纸张底
- **左对齐巨字标题**：标题不再居中，改为强有力的左对齐大字，作为整张海报的视觉锚点
- **强调色短线 + 编号要点**：标题下一道强调色短线；正文里「一、二、三、」序号自动变成强调色，形成清晰的编号列
- **4 个版式全部重写**为新的设计语言（经典公文 / 横幅庆典 / 竖排侧栏 / 极简留白），保留版式选择多样性

## 🌈 全新配色系统：强调色 + 底色，可任意搭配

- **强调色方案（6 种）**：朱红、赭石、靛蓝、松绿、墨黑、紫绛 —— 控制短线、编号、印章、小标题的颜色
- **底色（6 种）**：暖纸、米白、纯白、浅青、浅墨、象牙 —— 独立可选，与强调色自由组合，适配不同主题

## 🔤 字体优化：4 款真实可加载中文字体

精简并替换为 Google Fonts 可真正加载的字体，解决旧版「小标宋 / 楷体 / 仿宋在网页上加载不到、统一回落系统字体」的问题：

- 宋体巨字（思源宋体）｜小薇刊头（ZCOOL XiaoWei）｜思源黑体｜青铜黄油（ZCOOL QingKe HuangYou）

## ✏️ 富文本编辑器：完全重构

- **浮动气泡工具条**：选中文字即弹出加粗 / 颜色快捷条，不再「点工具栏就丢光标」
- **分组工具栏**：格式 / 对齐 / 标题层级 / 颜色色板，分组清晰，移动端可横滑
- **字数统计**底栏 + 当前格式高亮
- 编辑区可纵向滚动、可拖拽调整高度

## 🔖 印章：文字可编辑

- 修复了旧版印章文字写死「横税纪检」、设置项形同虚设的问题
- 印章文字现在 **2–6 字可自由编辑**，颜色跟随当前强调色

## 🐛 问题修复

- **正文大标题不再消失**：修复旧版用大标题格式后预览中文字被隐藏的 bug
- **移除底部落款行**、页眉改为纯中文
- 修复字体配置的循环导入、移除冲突的依赖配置
- 新增防缓存，避免更新后看到旧版本

## 📌 说明

- 节日自动推荐系统（春节 / 清明等）本次暂停，入口已隐藏，相关数据保留待后续启用
- 旧的纹理背景、装饰元素功能保留可用，但新设计默认不启用
