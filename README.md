# 警示海报生成器（Reminder Poster Generator）

一个纯前端、零后端依赖的中文警示/提醒海报生成工具，面向党建、纪检、安全生产、节日廉洁提醒等场景。输入标题和正文，即可生成可下载的竖版海报，支持廉洁主题纹样、多种版式与中式装饰边框，并对移动端做了专项优化。

> 纯客户端运行，内容不经过任何服务器，历史记录保存在浏览器本地。
>
> 🌐 在线体验：<https://hsjj.netlify.app>

## 功能特性

### 廉洁主题装饰系统
- **10 种中式经典纹样**（SVG 精绘，非 emoji）：
  - 廉洁元素：**莲花水印**（背景隐约可见的大莲花）、**青竹**（虚心有节）、**清水**（一清如水）、**幽兰**（君子之风）
  - 传统吉祥：祥云纹、缠枝莲、如意角、四角宝相、连珠边、花纹边框
- **默认装饰组合**：莲花水印 + 青竹 + 清水 + 四角宝相（一键开箱即用的廉洁视觉）
- 多种装饰可叠加开关，自由组合

### 海报样式
- **8 种装饰边框**：双线装裱、简约单线、三线套框、云纹角花、缠枝角花、四角印章、回纹套框、无边框
- **6 种强调色方案**：朱红、赭石、靛蓝、松绿、墨黑、紫绛（控制标题、短线、编号、印章颜色）
- **6 种底色**：暖纸、米白、象牙（暖系）+ 浅青、浅墨、浅艾（冷系），每种配独立的条目卡片色
- **4 种版式**：经典公文、横幅庆典、竖排侧栏、极简留白
- **5 款公文标准字体**（标题与正文可分别设置）：方正小标宋、宋体、楷体、仿宋、行楷
- **宣纸材质层**：宣纸纤维点阵 + 水墨角晕，强度可调（0–100%）
- 可调标题/正文字号、画布尺寸，提供尺寸预设（朋友圈竖图、公众号方图、A4 打印）

### 模板库（三页签）
- **编辑设计 / 模板库 / 历史记录** 三页签切换
- **模板库**：
  - 临近节日情景区（自动检测，一键套用）
  - 通用模板（内置多个党建/纪检/安全场景文案）
  - 节日模板（7 大节日分组折叠，各含专属文案）
  - **我的模板**：本地保存的自定义模板，支持编辑、删除
- **本地覆写**：内置模板只读，可"存为我的模板"后自由编辑
- 套用任何模板都**保留默认视觉骨架**（装饰 + 边框），只改文案与配色

### 节日主题系统
- 内置 **7 大法定节日**：元旦、春节、清明、劳动节、端午、中秋、国庆
- 每个节日含专属配色方案、底色、印章文字、版式、文案模板
- 内置 **2024–2030 农历查表**，自动检测临近节日，编辑页顶部推荐横幅 + 一键套用

### 编辑器体验
- **亮色 / 暗色主题切换**：右上角浮动按钮，默认亮色（象牙白），选择持久化
- 富文本编辑器：加粗、斜体、下划线、对齐、标题层级、文字颜色、重点遮挡
- 浮动气泡工具条：选中文字即弹出快捷格式条，修复了点击工具栏丢光标的问题
- 正文条目化：带编号（一、二、）的段落自动渲染为带左侧强调色竖条 + 浅底的卡片

### 移动端优化
- 富文本编辑器工具栏单行可横滑，触摸目标 ≥ 40px
- 预览页控制项**折叠分组**，减少滚动
- 尺寸/字号滑块带**数值输入框**，方便精确调整
- 全屏预览支持**双指缩放 + 拖拽**
- 下载按钮带 loading 反馈，针对 iOS Safari 的 canvas 内存上限做了自适应缩放

### 其他
- 历史记录（最多 10 条），自动保存到 `localStorage`，支持加载/删除，带配额自动管理
- 下载导出前等待字体加载完成，避免标题回落系统字体

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | React 18 + TypeScript（strict 模式） |
| 构建 | Vite 5 |
| 样式 | Tailwind CSS（CDN） |
| 字体 | 系统公文标准字体优先（方正小标宋/STZhongsong/楷体/仿宋），Google Fonts 兜底 |
| 导出 | `html2canvas`（将 DOM 海报道具截图为 PNG） |
| 持久化 | 浏览器 `localStorage` |

> 本项目最初基于 Google AI Studio（Gemini）构建，已移除 AI 依赖，改为完全本地生成。

## 本地运行

**前置要求**：Node.js（建议 18+）

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

浏览器打开终端输出的地址（默认 `http://localhost:5173`）即可。
手机可在同一局域网下访问终端显示的 Network 地址。

## 构建与部署

```bash
# 类型检查 + 生产构建，产物输出到 dist/
npm run build

# 本地预览构建产物
npm run preview
```

### 部署到 Netlify / Vercel
本项目无需额外配置：
- **Build Command**：`npm run build`
- **Output Directory**：`dist`

仓库已含 `netlify.toml`（构建命令、发布目录、SPA 重定向）。推送到 GitHub 主分支后自动触发部署。

## 使用说明

1. **编辑内容**：输入标题，在富文本编辑器中撰写正文（可使用工具栏设置格式）
2. **快速开始**：切换到「模板库」页签，选一个模板套用文案，或点顶部节日推荐横幅一键套用节日主题
3. **生成海报**：点击「开始生成海报」进入预览
4. **调整样式**：在折叠面板中切换版式、强调色、底色、字体、装饰、边框、纸张质感强度等
5. **下载**：点击「下载海报图片」，自动保存为 PNG

## 设计说明

- 所有背景纹理与装饰元素均采用**纯 CSS 渐变 + 内联 data-URI SVG** 实现，刻意避免 SVG filter 和跨域图片，规避 iOS Safari 下 `html2canvas` 导出时的安全错误。
- 印章使用 `mix-blend-mode: multiply` 模拟拓印按压效果，文字空时自动隐藏。
- 编辑器外壳采用 CSS 变量驱动的亮/暗双主题，切换 `<html data-ui-theme>` 即可全局换肤。
